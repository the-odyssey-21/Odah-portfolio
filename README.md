Cybersecurity Portfolio

A modern, cyberpunk-inspired personal portfolio showcasing my journey, skills, and projects in **Cybersecurity & Penetration Testing**.

About The Project

This is a fully responsive, single-page portfolio website designed with a **hacker aesthetic** and built using pure HTML, CSS, and JavaScript.

It highlights:

* My cybersecurity background
* Core technical skills
* Featured projects
* Blog-style writeups
* Contact links

---

Built With

* HTML5
* CSS3 (custom properties + animations)
* Vanilla JavaScript
* Google Fonts (JetBrains Mono)

---

Features

* Matrix-style binary rain animation
* Typewriter text effect
* Smooth scroll & reveal animations
* Cyberpunk neon UI (green/blue theme)
* Fully responsive design
* Interactive project cards
* Minimal & fast (no frameworks)
